# 🚗 Vehicle Number Plate Detection System

## 📌 Overview
This project is an AI-based system that detects and recognizes vehicle number plates using image processing techniques.

## 🎯 Objective
The main objective of this project is to automatically identify vehicle number plates from images or video streams.

## 🛠️ Technologies Used
- Python
- OpenCV
- EasyOCR / Tesseract OCR

## ⚙️ How it Works
1. Capture image from camera or input image
2. Detect number plate region using OpenCV
3. Extract text using OCR
4. Display the detected number

## 📷 Output
The system highlights the number plate and shows the extracted text.

## 🚀 Applications
- Traffic monitoring
- Parking management
- Toll collection systems
- Security systems

## 🔮 Future Scope
- Real-time video processing
- Integration with databases
- Improved accuracy using deep learning
